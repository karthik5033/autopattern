# .agent — Developer Workflows

This folder contains instructions and workflows for AI coding agents
(Copilot, Codex, Claude Code, Cursor, etc.) working on the AutoPattern codebase.

## Files

| File | Purpose |
|---|---|
| `AGENTS.md` | Project-wide rules, architecture, conventions |
| `workflows/setup.md` | First-time environment setup |
| `workflows/add-feature.md` | Implementing a new feature end-to-end |
| `workflows/fix-bug.md` | Debugging and fixing issues |
| `workflows/release.md` | Building, testing, and publishing a release |
| `workflows/extension.md` | Working on the Chrome extension |
